# 台股盤前趨勢預估

每日盤前跑一次，依「隔夜美股 + 台股籌碼」規則引擎，預估當日台股方向（**偏多 / 偏空 / 震盪**）。

> ⚠️ 本工具為決策輔助，輸出為機率性方向預估，**非投資建議**。市場有大量隨機性與突發事件。

線上網址：https://tomkao761116.github.io/tw-stock-trend/

## 因子與權重（大盤）

| 因子 | 來源 | 權重 | 說明 |
|------|------|------|------|
| 台指期夜盤 | FinMind | 2.5 | 隔夜訊息對台股最直接的反映 |
| 費半 SOX | Yahoo Finance | 2.5 | 與台股半導體連動最強 |
| 台積電 ADR | Yahoo Finance | 2.5 | 直接影響台積電開盤 |
| 那斯達克 | Yahoo Finance | 2.0 | 科技股氣氛 |
| 外資買賣超 | FinMind | 2.0 | 前一交易日 |
| 外資期貨 | FinMind | 2.0 | 未平倉淨額變化 |
| 台幣匯率 | Yahoo Finance | 1.0 | 升值 = 外資進場訊號 |
| VIX 恐慌 | Yahoo Finance | 2.0 | 只貢獻負分 |
| 融資餘額 | FinMind | 1.0 | 反向：散戶槓桿過熱訊號 |

每因子先標準化為 [-1, +1]，乘權重後加總：總分 ≥ +4 → 偏多｜總分 ≤ -4 → 偏空｜其餘 → 震盪。

## 細分類別（大盤以外）

除了大盤，同一套引擎會額外算出獨立方向預估的子類別，各自有專屬因子組、權重與門檻：

| 類別 | 因子 | 回測對照標的 |
|------|------|------|
| **科技股/半導體** | 台指期夜盤、SOX、台積電ADR、那斯達克（沿用大盤因子）、**輝達NVDA**（AI需求端訊號） | 0052.TW（富邦科技） |
| **金融股** | 美股金融類股(XLF)、美債10年殖利率、台幣匯率、外資買賣超 | 0055.TW（元大MSCI金融） |
| **傳產/原物料/航運** | WTI原油、銅期貨、美元指數、航運ETF(BDRY) | 長榮/中鋼/台塑等權重組合 |

> ⚠️ **已知限制**：金融股、傳產類別沒有台股本地籌碼資料可用，只能靠美股對應類股與大宗商品做隔夜訊號，
> 準確度天生比大盤/科技股（有夜盤直接反映）弱，門檻也設得較保守。

## ETF 分頁（巢狀子分頁）

ETF 是「投資風格」維度，跟大盤/科技股/金融股/傳產這種「產業」維度不是同一件事，所以獨立成一個頂層分頁，底下再分三個子分頁：

| 子分頁 | 因子 | 回測對照標的 |
|--------|------|------|
| **股息型** | 美債殖利率（反向，債券替代效應）、美股金融類股、美股公用事業ETF(XLU)、外資買賣超、台幣匯率 | 0056.TW（元大高股息） |
| **成長型** | *不重新運算*——直接借用「科技股/半導體」分頁的結果 | 同科技股/半導體 |
| **債券型** | TLT（美國20年期公債，直接代理）、美債殖利率（反向）、VIX（避險效應）、MOVE指數（債市專屬波動率） | 00679B.TWO（元大美債20年） |

> 市值型（如 0050）不單獨開分頁：0050 台積電權重與大盤相近，套用同一套因子只是大盤的翻版，沒有新的判斷依據。
> 主動型 ETF 也不做——持股由經理人自由裁量、缺乏公開即時持股，無法設計穩定的隔夜訊號。
